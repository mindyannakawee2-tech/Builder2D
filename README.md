
# Builder2D

A lightweight SDL2-based 2D game framework for Python.

## Install

```bash
pip install .
```

## Features

- Scene / GameObject / Script
- Rigidbody2D + BoxCollider2D physics
- PNG/BMP sprite rendering
- Frame and spritesheet animation
- Basic UI: Canvas, Panel, Label, Button
- Audio: SoundEffect, Music
- Borderless window and fullscreen-desktop

## Demo

```bash
python demo.py
```
